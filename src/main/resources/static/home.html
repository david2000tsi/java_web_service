Basic Java Web Service

This is a basic Java application with web services using Spring Boot, it was developed using Eclipse IDE and Java 8.

You can see readme file to more info.

Available routes:

  • /gettimestamp
  • /makesum/<value1>/<value2>
  • /makesub/<value1>/<value2>
  • /makemul/<value1>/<value2>
  • /makediv/<value1>/<value2>
  • /message/save/<message>
  • /message/get
  • /message/update/<id>/<new_message>
  • /message/delete/<id>
